limesurvey plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/myusername/tutor-contrib-limesurvey

Usage
-----

::

    tutor plugins enable limesurvey
